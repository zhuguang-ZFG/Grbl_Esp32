# 换纸逻辑说明

## 硬件布局与基本特性

### 电机位置关系
```
进纸电机 ---- 纸张传感器 ---- 夹紧电机 + 面板电机
       (相距3cm)      (相距3cm)
```

### 基本状态说明
- **夹紧电机默认状态**：电机使能不运动即为夹紧状态
- **传感器位置**：位于夹紧电机与面板电机之间
- **传感器功能**：感应纸张存在和距离测量

### 电机运行限制
- **面板电机**：一般情况下不反转，仅用于位置判断时反转
- **进纸器电机**：永远不会反转
- **夹紧电机**：正转1.5cm即为松开夹子，反转到初始位置即为夹紧状态

## 进纸流程详细说明

### 工作步骤

1. **启动进纸**
   - 进纸电机开始运动
   - 等待纸张感应器检测到纸张

2. **检测到纸张后**
   - 夹紧电机运行（初始为夹紧态）→ 松开夹子
   - 面板电机同步运行
   - 纸张经过传感器后继续运行3.5cm

3. **夹紧行程**
   - 夹紧电机夹紧纸张
   - 进纸电机、面板电机同速快速运行
   - 运行约A4纸张行程，直到纸张脱离传感器

4. **位置校准**
   - 进纸电机停止
   - 面板电机反转
   - 直到传感器再次感应到纸张
   - 面板电机正转3cm左右（**关键步骤**，确保纸张位置正确）

5. **完成**
   - 纸张到达正确位置
   - 开始执行正常写字任务

### 重要说明
- **传感器触发后的5cm运行期间**：夹紧电机保持松开状态
- **纸张脱离传感器**：进纸电机停止信号
- **位置校准**：反转后再正转3cm是确保位置正确的关键

## 出纸流程详细说明

### 工作步骤

1. **检测纸张位置**
   - 面板电机反转
   - 直到传感器感应到纸张后停止

2. **完全出纸**
   - 面板电机快速正转
   - 运行距离：A4纸张长度 + 3-5cm
   - 确保纸张完全送出

3. **停止**
   - 面板电机停止
   - 出纸流程完成

#### 通信协议时序
1. **锁存阶段**：拉低STCP，准备数据移入
2. **移位阶段**：8个时钟周期，从MSB到LSB逐位移入数据
3. **数据建立**：每个时钟上升沿前数据线必须稳定1μs
4. **锁存输出**：拉高STCP，8位数据同时并行输出到Q0-Q7
5. **硬件稳定**：等待10μs确保输出信号稳定
