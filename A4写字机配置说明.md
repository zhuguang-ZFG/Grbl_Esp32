# A4写字机配置说明

## 硬件规格

### 机械行程
- **X轴行程**: 210mm (A4纸张宽度)
- **Y轴行程**: 297mm (A4纸张高度) 
- **Z轴行程**: 20mm (抬落笔控制)

### 坐标系设置
- **原点位置**: 左下角 (0, 0)
- **X轴**: 从左到右 (0 → 210mm)
- **Y轴**: 从下到上 (0 → 297mm)
- **Z轴**: Z=0落笔写字，Z=20抬笔移动

### 建议书写区域
```
有效书写区域: 200mm × 287mm (预留边距)
左边距: 10mm
右边距: 0mm (利用到210mm)
下边距: 10mm  
上边距: 0mm (利用到297mm)
```


## 使用方法

### 1. 初始化设置
```gcode
G21                     ; 单位设为毫米
G90                     ; 绝对坐标模式
G92 X0 Y0 Z20           ; 设置原点在左下角，抬笔位置
```

### 2. 基本书写流程
```gcode
; 示例: 在A4纸张中心写字
G0 X105 Y148.5 Z20     ; 移动到中心位置，抬笔
G0 X105 Y148.5 Z0      ; 落笔
; ... 开始书写 ...
G0 X105 Y148.5 Z20     ; 抬笔
```

### 3. 边界避让
```gcode
; 安全书写区域
G0 X10 Y10 Z0          ; 左下角最小坐标
G0 X200 Y287 Z0        ; 右上角最大坐标
```

### 4. M0换纸测试
```gcode
G0 X100 Y100 Z20       ; 移动到安全位置
M0                      ; 触发M0，回原点发送ok
G1 X150 Y150 Z0        ; 这个命令应该触发自动换纸
```

## 注意事项

### 1. 安全边距
- 建议保留5-10mm边距避免书写到纸张边缘
- 确保换纸机构不干扰书写区域

### 2. 速度设置
- 高速书写时注意电机发热
- 复杂图形建议降低速度

### 3. 笔具适配
- 确保笔具在Z=0位置能稳定书写
- 调整抬笔高度避免拖笔

### 4. 纸张定位
- 确保每次换纸后纸张位置一致
- 传感器精度影响最终书写质量

## 常见问题

### Q: 书字机原点在哪里？
A: 左下角(0,0)，Z轴原点在纸面上方0mm

### Q: 如何处理A4纸张边距？
A: 建议使用X:10-200mm, Y:10-287mm的书写区域

### Q: M0换纸如何触发？
A: M0后需要有后续G-code命令才会触发换纸

### Q: 如何验证A4覆盖？
A: 运行a4_paper_test.nc文件进行全面测试
