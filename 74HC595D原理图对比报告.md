## ✅ 5. 验证建议

### 5.1 硬件验证步骤

1. **控制信号验证**:
   ```
   使用万用表测量：
   - ESP32 GPIO32 ↔ 74HC595D Pin 14 (SI)
   - ESP32 GPIO3 ↔ 74HC595D Pin 11 (SRCLK)
   - ESP32 GPIO5 ↔ 74HC595D Pin 12 (RCLK)
   ```

2. **输出信号验证**:
   ```
   使用逻辑分析仪或示波器：
   - 发送测试数据，观察Q0-Q7输出
   - 验证位映射是否正确
   ```

3. **功能验证**:
   ```
   使用调试命令：
   M806 0x01  ; 仅Q0高电平，检查LED
   M806 0x02  ; 仅Q1高电平，检查使能
   M806 0x04  ; 仅Q2高电平，检查夹紧方向
   ```

### 5.2 原理图查看建议

1. **打开原理图文件**:
   - 使用Protel/Altium Designer打开 `MK38-总图.Sch`
   - 查找U5 (74HC595D)组件

2. **检查连接**:
   - 查看U5 Pin 14连接到哪个ESP32引脚
   - 查看U5 Pin 11连接到哪个ESP32引脚
   - 查看U5 Pin 12连接到哪个ESP32引脚
   - 查看U5输出引脚（QA-QH）连接到哪些组件

3. **验证输出连接**:
   - U5-QA (Q0) → LED或按钮指示灯
   - U5-QB (Q1) → HR4988使能引脚
   - U5-QC到U5-QH → 各个HR4988驱动器的DIR/STEP引脚
