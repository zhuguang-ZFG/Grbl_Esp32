## 🔍 3. 代码中的潜在问题

### 3.1 引脚冲突检测代码问题

**文件**: `Grbl_Esp32/src/PaperChangeHardware.cpp` 第109行

```cpp
#ifdef Z_STEP_PIN
if (HC595_LATCH_PIN == Z_STEP_PIN)  // ⚠️ 缺少大括号
    grbl_sendf(...);
    pin_conflict = true;  // 这行代码不在if语句内！
}
```

**问题**: 
- ⚠️ **语法错误** - if语句缺少大括号
- 这会导致 `pin_conflict = true;` 总是执行，即使没有冲突

**修复建议**:
```cpp
#ifdef Z_STEP_PIN
if (HC595_LATCH_PIN == Z_STEP_PIN) {
    grbl_sendf(CLIENT_ALL, "[MSG: Error: HC595_LATCH_PIN (%d) conflicts with Z_STEP_PIN (%d)]\n", HC595_LATCH_PIN, Z_STEP_PIN);
    pin_conflict = true;
}
```
