# 自动换纸系统说明

## 系统概述

本写字机集成了完整的自动换纸系统，通过74HC595D移位寄存器控制多个步进电机，实现无人值守的连续书写操作。

## 硬件配置

### 主控制器
- **ESP32-WROOM-32E**: 40MHz晶振，主控芯片
- **HR4988**: 4个步进驱动器，驱动X/Y/Z轴和换纸系统

### 74HC595D控制引脚
```
功能           GPIO    引脚    说明
数据输入(DS)   GPIO16   P27    串行数据
移位时钟(SHCP) GPIO17   -      移位寄存器时钟
锁存时钟(STCP) GPIO5    -      存储寄存器时钟
纸张传感器     GPIO34   P6     纸张检测(输入)
```

### 74HC595D输出映射
```
HC595引脚  功能           电机/控制
Q0 (pin15)  一键换纸按钮   常开按钮接VCC
Q1 (pin1)   未使用        -
Q2 (pin2)   压纸抬落方向  压纸电机
Q3 (pin7)   压纸抬落步进  压纸电机
Q4 (pin4)   出纸面板方向  面板电机
Q5 (pin5)   出纸面板步进  面板电机
Q6 (pin6)   进纸器方向    进纸电机
Q7 (pin7)   进纸器步进    进纸电机
```

## 换纸工作流程

### 自动换纸 (M0指令触发)
1. **M0指令处理**: 回到原点，立即发送"ok"确认M0完成
2. **内容检测**: 2秒内检查M0后是否有后续G-code指令
3. **状态等待**: 系统等待下一个G-code命令到达
4. **换纸触发**: 收到下一个命令时，如果有待处理换纸则启动
5. **出纸阶段**: 面板电机送出写完纸张 (100步)
6. **进纸阶段**: 进纸器送入新纸，监测传感器
7. **传感器触发**: 纸张接触传感器，启动夹纸
8. **夹纸动作**: 压纸电机+面板电机同时启动 (20步)
9. **释放检测**: 纸张离开传感器，所有电机停止
10. **精确定位**: 延时1秒，面板电机前进32步后退2步
11. **完成回复**: 换纸完成，继续执行触发命令

### 工作流程示例
```gcode
G0 X10 Y10 Z20     ; 移动到安全位置
M0                  ; 回原点，发送ok，等待下一个命令
                    ; 上位机收到ok，发送下一个命令
G1 X20 Y20 Z0      ; 触发待处理的换纸，换纸完成后执行
```

### 一键换纸 (按钮触发)
- **短按(<2秒)**: 立即启动完整换纸流程
- **长按(≥2秒)**: 紧急停止所有换纸操作

## 配置参数

### 电机参数
```cpp
// 进纸器电机
FEED_STEPS_PER_MM:    100.0
FEED_MAX_RATE:        3000.0 mm/min
FEED_MAX_TRAVEL:      300.0 mm

// 出纸面板电机  
PANEL_STEPS_PER_MM:   80.0
PANEL_MAX_RATE:       2000.0 mm/min
PANEL_MAX_TRAVEL:     50.0 mm

// 压纸抬落电机
CLAMP_STEPS_PER_MM:   150.0
CLAMP_MAX_RATE:       1500.0 mm/min
CLAMP_MAX_TRAVEL:     10.0 mm
```

### 时序参数
```cpp
M0_CONTENT_CHECK_TIMEOUT_MS:  2000    // M0内容检测超时
PAPER_FEED_TIMEOUT_MS:         5000    // 进纸超时
PAPER_BUTTON_DEBOUNCE_MS:      50      // 按钮防抖
PAPER_BUTTON_LONG_PRESS_MS:      2000    // 长按检测
HC595_UPDATE_DELAY_US:          10      // HC595更新延时
```

## 使用方法

### 1. G-code编程
```gcode
G0 X10 Y10 Z20     ; 移动到安全位置
M0                  ; 触发换纸(如果有后续内容)
G1 X20 Y20 Z0      ; 继续书写
```

### 2. 手动操作
- **短按换纸按钮**: 立即换纸
- **长按换纸按钮**: 紧急停止

### 3. 状态监控
系统会实时输出状态信息：
```
[MSG: Smart M0 detected, checking for content...]
[MSG: Paper change state: EJECTING]
[MSG: Paper detected, engaging clamp]
[MSG: Paper change completed successfully]
```

## 故障排除

### 常见问题
1. **传感器无响应**: 检查GPIO34连接和传感器供电
2. **电机不转动**: 检查HC595D连接和驱动器使能
3. **按钮无反应**: 检查按钮连接和Q0输出
4. **换纸不完整**: 检查步进参数和超时设置

### 调试命令
```gcode
$Status             ; 查看系统状态
$Help               ; 查看帮助信息
$$                  ; 查看所有设置
```
