## 🎯 核心技术改进

### 1. HR4988 VREF统一控制系统 ✅

**技术实现**:
- **控制引脚**: GPIO25 (ESP32 DAC输出)
- **控制方式**: 8位DAC (0-255 → 0-3.3V)
- **检测电阻**: 0.05Ω (重要！非0.1Ω)
- **电流公式**: `I = V_REF / (8 × 0.05Ω)`

**关键文件**:
- `Grbl_Esp32/src/HR4988VREF_SIMPLE.h`
- `Grbl_Esp32/src/HR4988VREF_SIMPLE.cpp`
- `Grbl_Esp32/src/Machines/custom_3axis_hr4988.h`

**技术特性**:
- 多级电流模式：待机(0.3A)、精密(0.1A)、正常(1.0A)、高扭矩(1.5A)、最大(2.0A)
- 自适应控制：根据步进频率动态调整
- 温度保护：高温自动降流
- 统一控制：7个HR4988驱动器共享同一VREF

### 2. 双重互斥保护机制 ✅

**设计目标**: 同一时间只能有一组电机工作，避免冲突和电源过载

**第一层 - 硬件互斥 (GPIO26)**:
```
GPIO26 = LOW  → 写字模式 (XYZ轴使能，换纸电机禁用)
GPIO26 = HIGH → 换纸模式 (XYZ轴禁用，换纸电机使能)
```

**第二层 - 软件互斥 (HC595 Q1)**:
```
Q1 = HIGH → 换纸电机软件禁用，XYZ轴可以工作
Q1 = LOW  → 换纸电机软件使能，XYZ轴禁用
```

**安全特性**:
- 模式切换延时：100ms
- 状态一致性检查
- 上电默认：所有电机禁用

### 3. 换纸系统9状态机设计 ✅

**状态定义**:
1. `IDLE` - 空闲状态
2. `PRE_CHECK` - 预检查
3. `EJECTING` - 退出纸张
4. `FEEDING` - 进纸
5. `UNCLAMP_FEED` - 松开进纸
6. `CLAMP_FEED` - 夹紧进纸
7. `FULL_FEED` - 完全进纸
8. `REPOSITION` - 重新定位
9. `COMPLETE` - 完成

**核心文件**:
- `Grbl_Esp32/src/PaperChange.cpp`
- `Grbl_Esp32/src/PaperChangeLogic.cpp`
- `Grbl_Esp32/src/PaperChangeHardware.cpp`
- `Grbl_Esp32/src/PaperChangeTypes.h`

**安全机制**:
- 状态超时保护：30秒
- 步数限制：防止越界
- 错误恢复：自动重试+手动介入

### 4. 硬件配置规范 ✅

**电机驱动配置**:
```
XYZ轴: 4个HR4988驱动器
换纸系统: 3个HR4988驱动器  
总计: 7个HR4988驱动器统一控制
```

**关键GPIO定义** (custom_3axis_hr4988.h):
```cpp
#define HR4988_VREF_PIN        GPIO_NUM_25    // VREF控制
#define STEPPERS_DISABLE_PIN   GPIO_NUM_4     // XYZ轴使能控制
#define PAPER_MOTORS_ENABLE    GPIO_NUM_26    // 换纸电机硬件互斥
```

**互斥控制**:
- GPIO26 + HC595 Q1 双重保护
- 传感器去抖动：50ms
- 按钮检测：GPIO35，2秒长按
