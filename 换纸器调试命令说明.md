# 换纸器调试命令使用说明

## 概述
Grbl_Esp32集成了自动换纸功能，并提供了一系列调试命令用于手动控制和测试各个组件。

## 调试命令 (M100-M114)

### 单步调试命令 (M100-M106)

### 1. M100 - 进纸电机调试
```
M100 S[步数] 
```
- 功能：控制进纸电机转动指定步数
- 参数：S - 步数（正数前进，负数后退）
- 示例：M100 S800  （进纸800步）

### 2. M101 - 面板电机调试  
```
M101 S[步数]
```
- 功能：控制面板电机转动指定步数
- 参数：S - 步数（正数前进，负数后退）
- 示例：M101 S160  （面板前进160步）

### 3. M102 - 夹具电机调试
```
M102 S[步数]
```
- 功能：控制夹具电机转动指定步数
- 参数：S - 步数（正数夹紧，负数松开）
- 示例：M102 S100  （夹具动作100步）

### 4. M103 - 读取所有传感器状态
```
M103
```
- 功能：显示所有传感器的实时状态
- 输出信息包括：
  - 纸张传感器（PAPER_SENSOR_PIN）
  - 限位传感器（PAPER_LIMIT_PIN）  
  - 按钮状态（PAPER_BUTTON_PIN）
  - 紧急停止状态
  - HC595D输出状态

### 5. M104 - 紧急停止所有电机
```
M104
```
- 功能：立即停止所有换纸器电机并设置紧急停止标志
- 所有电机将被禁用，需要使用M105重置

### 6. M105 - 重置紧急停止
```
M105
```
- 功能：清除紧急停止标志，恢复电机控制功能
- 在紧急停止后必须执行此命令才能继续操作

### 7. M106 - HC595D直接控制
```
M106
```
- 功能：直接设置HC595D输出为0xFF（所有电机使能）
- 用于测试HC595D芯片和驱动电路

## 完整换纸流程命令 (M110-M114)

### 8. M110 - 启动完整换纸流程
```
M110
```
- 功能：执行完整的7步自动换纸流程
- 流程包括：
  1. 进纸动作 → 纸张传感器检测
  2. 夹具动作 → 固定纸张  
  3. 纸张离开检测 → 释放夹具，确保纸张完全就位
  4. 面板电机微调 → 前进32步后后退2步，精确定位
  5. 走固定步数 → 面板电机完成最终定位
  6. 系统就绪 → 恢复正常写字功能
  7. 换纸完成 → 返回初始状态

### 9. M111 - 一键换纸模式
```
M111
```
- 功能：简化的一键式换纸操作
- 适用于快速换纸场景，流程更简化

### 10. M112 - 紧急停止换纸
```
M112
```
- 功能：立即停止换纸流程并进入紧急状态
- 比M104更彻底的停止，影响整个换纸状态机

### 11. M113 - 重置换纸系统
```
M113
```
- 功能：重置换纸系统到初始状态
- 在错误或紧急停止后使用，清除所有状态

### 12. M114 - 查询换纸状态
```
M114
```
- 功能：显示当前换纸系统的状态
- 输出信息：
  - 换纸系统是否激活 (ACTIVE/INACTIVE)
  - 当前状态名称（如：FEEDING、CLAMPING等）

## 硬件配置参数

### 电机精度
- 面板电机：80步/mm
- 进纸电机：根据实际机械传动确定
- 夹具电机：根据夹具机械确定

### GPIO引脚定义
- HC595_DATA_PIN: GPIO32  (74HC595D DS引脚)
- HC595_CLOCK_PIN: GPIO3  (74HC595D SHCP引脚)
- HC595_LATCH_PIN: GPIO2  (74HC595D STCP引脚)
- STEPPERS_DISABLE_PIN: GPIO4  (总使能控制)

### 传感器引脚
- PAPER_SENSOR_PIN: 纸张检测传感器
- PAPER_LIMIT_PIN: 限位传感器
- PAPER_BUTTON_PIN: 手动按钮

## 调试建议流程

1. **连接检查**：
   ```
   M103  ; 检查传感器初始状态
   M106  ; 测试HC595D输出
   ```

2. **电机测试**：
   ```
   M100 S100  ; 测试进纸电机
   M101 S100  ; 测试面板电机  
   M102 S100  ; 测试夹具电机
   ```

3. **传感器验证**：
   ```
   M103  ; 手动触发传感器后再次检查
   ```

4. **紧急功能测试**：
   ```
   M104  ; 触发紧急停止
   M105  ; 重置紧急停止
   ```

5. **完整流程测试**：
   ```
   M114  ; 查看换纸状态
   M110  ; 启动完整换纸流程
   M114  ; 监控流程状态
   ```

6. **快速换纸测试**：
   ```
   M111  ; 一键换纸模式
   M114  ; 确认换纸完成
   ```

## 注意事项

1. 使用调试命令前确保电机连接正确
2. 步数参数建议从小到大测试
3. 如遇异常立即使用M104紧急停止
4. HC595D控制会影响所有电机，谨慎使用
5. 传感器状态会实时显示，便于故障诊断

## 自动换纸流程

完整的自动换纸流程可通过主命令触发（具体命令参考相关文档），包含7个步骤：
1. 进纸 → 纸张传感器检测
2. 夹具动作 → 固定纸张
3. 面板精确定位 → 32步前进+2步后退
4. 固定步数前进 → 完成就位
5. 系统就绪 → 恢复写字功能

调试命令可用于单独测试流程中的各个步骤。
